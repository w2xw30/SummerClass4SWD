# Web Development (Level 4) — Weekly Tasks
Herald College / University of Wolverhampton

This repository contains my weekly tasks and exercises completed during the Web Development module. Each week has its own folder with the work done in tutorials and workshops.

## Weekly Topics

- **Week 1** - HTML & CSS static site, Git basics, GitHub Pages deployment
- **Week 2** - CSS animations, responsive design, JavaScript DOM manipulation
- **Week 3** - Git branching, pull requests, merge conflicts, documentation
- **Week 4** - Node.js, Express server, REST API endpoints
- **Week 5** - Connecting frontend to backend using fetch()
- **Week 6** - SQLite database, Sequelize ORM, CRUD operations
- **Week 7** - Deployment, basic authentication, project presentation
