All About Tomatoes

Tomatoes are vibrant, juicy, and one of the world's most versatile fruits used in cooking. Below are examples of how we display images of tomatoes using HTML.

Example 1: Specifying Dimensions

This image uses the path images/tomato.jpg. We have explicitly set the width to 300 pixels and the height to 200 pixels using HTML attributes.

Example 2: Relative Path & Alt Text Importance

This image uses the path ./tomatoes.jpg (the ./ explicitly means "look in the current folder"). If the image fails to load, the browser will display the alt text instead so screen readers and users still know what belongs here.
